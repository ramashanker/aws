# Important aws kms command

### Create KMS KEY
``` 
aws kms create-key
```

### Encrypt a key
``` 
aws kms encrypt
```
### Decrypt a key

``` 
aws kms decrypt
```
### Re-Encrypt a key

``` 
aws kms re-encrypt
```
### Turn on automatic key rotation(Only for symmetric keys)

``` 
aws kms enable-key-rotation
```
